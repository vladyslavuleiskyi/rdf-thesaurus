# SMBA Thesaurus

A machine-readable semantic model that brings together Business Analysis terminology from multiple authoritative sources into a single, navigable knowledge graph.

## About

Business Analysis knowledge is scattered across different standards â€” BABOK, PMI-PBA, IREB, ISO and others. Each uses slightly different terminology and definitions. This project consolidates terms into a unified SKOS thesaurus with clear relationships and source attribution.

## Tech stack

- Astro (static site generation)
- RDF/SKOS (semantic web standards)
- Turtle format for term definitions
- vis-network for graph visualization

## Live site

https://thesaurus.smba.ai

## What you can do

- Browse terms with definitions and relationships
- Explore the interactive knowledge graph
- Trace definitions back to source standards
- Export machine-readable RDF data
